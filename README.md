Portfolio Personal | Iñaki Arraztio
Este repositorio contiene el código fuente de mi portfolio profesional. Es una plataforma diseñada para centralizar mi trayectoria como Desarrollador de Software, exponiendo mis proyectos destacados, stack tecnológico y formación académica como Técnico Universitario en Desarrollo de Aplicaciones Informáticas.

🎯 Concepto y Diseño
El diseño fue concebido bajo una estética moderna y funcional, priorizando la accesibilidad a la información y una navegación simplificada.

Identidad Profesional: Uso de elementos visuales que refuerzan mi rol como Desarrollador Full Stack.
Call to Action (CTA): Acceso directo a mi Curriculum Vitae y canales de contacto para agilizar la comunicación con reclutadores y colaboradores.
Secciones Clave: Navegación fluida entre Proyectos, Habilidades, Sobre Mí y Contacto.
🛠️ Stack Tecnológico
Para la construcción de este sitio utilicé tecnologías que permiten un alto rendimiento y escalabilidad:

Core: React.js / JavaScript (ES6+).
Estilos: TailwindCSS / CSS3 (enfocado en diseño responsive).
Despliegue: Vercel (CI/CD integrado).
🚀 Características Técnicas
Arquitectura de Componentes: Estructura modular para facilitar el mantenimiento y la actualización de nuevos proyectos.
Optimización UX: Enfoque en la velocidad de carga y adaptabilidad para dispositivos móviles.
Integración Social: Enlaces directos a perfiles profesionales en LinkedIn y GitHub.
📍 Desarrollado en Tandil, Argentina.
